## Web tools and Library

🪄 [Optimo — Media Compression Commands](https://optimo.microlink.io/?utm_source=cocoweb.fr)
Effortless media optimizer for the web.
Free, open-source CLI that optimizes and converts images and videos using format-specific compression built on ImageMagick and FFmpeg.

☀️ [Radiant — Open Source Shaders & Effects](https://radiant-shaders.com/?utm_source=cocoweb.fr)
Production ready shaders & effects for the web. No dependencies. Just drop in.

🌍 [Shuding/cobe](https://github.com/shuding/cobe?utm_source=cocoweb.fr)
5KB WebGL globe lib.
Use any DOM element as bindable markers.CSS transitions, animations, filters, interactivity, all just work.
High perf, zero deps, ~5KB.
