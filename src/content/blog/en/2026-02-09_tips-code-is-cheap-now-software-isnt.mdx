<AnchorSection emoji="📗" id="how-to-structure-vue-projects">[How to Structure Vue Projects](https://alexop.dev/posts/how-to-structure-vue-projects/)</AnchorSection>

Very interesting article which compare 5 types of architecture for frontend project. Each architecture has their pros/cons and there are articles to learn more about each types.
Here are the 5 types of architectures :
* Flat structure for small projects
* Atomic Design for scalable applications
* Modular approach for larger projects
* Feature Sliced Design for complex applications
* Micro frontends for enterprise-level solutions
