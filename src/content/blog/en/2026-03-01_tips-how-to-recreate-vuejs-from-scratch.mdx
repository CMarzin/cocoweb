<h2 class="font-display text-2xl text-foreground">Web tools and Library</h2>

<WebLinkSection emoji="⚗️" url="https://github.com/Siddhant-K-code/distill" text="Distill" templateClass="mt-4">
**Context intelligence layer for AI agents.**
Deduplicates, compresses, and manages context across sessions - so your agents produce reliable, deterministic outputs. Includes a dedup pipeline with ~12ms overhead and persistent context memory with write-time dedup and hierarchical decay.
Less redundant data. Lower costs. Faster responses. Deterministic results.
</ WebLinkSection>

<WebLinkSection emoji="✈️" url="https://vizejs.dev/index.html" text="Vize JS" templateClass="mt-4">
Unofficial High-Performance Vue.js Toolchain in Rust
</ WebLinkSection>

<WebLinkSection emoji="💻" url="https://opentui.com/" text="Terminal UIs in TypeScript" templateClass="mt-4">
Build rich, interactive terminal interfaces with TypeScript bindings, first-class React/Solid support, and a C ABI for any language.
</ WebLinkSection>

<WebLinkSection emoji="🤖" url="https://lifo.sh/" text="Lifo.sh" templateClass="mt-4">
A browser-native operating system. Run agents securely on your machine — no VM, no cloud sandbox, no setup.
</ WebLinkSection>

<WebLinkSection emoji="🪄" url="https://github.com/nicobailon/visual-explainer?tab=readme-ov-file" text="Visual explainer" templateClass="mt-4">
An agent skill that turns complex terminal output into styled HTML pages you actually want to read.
</ WebLinkSection>

<WebLinkSection emoji="🍲" url="https://github.com/sanity-labs/react-logo-soup" text="React logo soup" templateClass="mt-4">
A tiny React library that makes logos look good together.
</ WebLinkSection>

<WebLinkSection emoji="🎯" url="https://github.com/pranshuparmar/witr" text="Why is this running" templateClass="mt-4">
WITR explains **where a running thing came from**, **how it was started**, and **what chain of systems is responsible for it existing right now**, in a single, human-readable output or an **interactive TUI dashboard**.
</ WebLinkSection>

<WebLinkSection emoji="🏰" url="https://effect.website/" text="Effect Typescript" templateClass="mt-4">
The best way to ship faster and safer in TypeScript. Instead of writing helpers and TS code at hand, use Effect to handle the heavy work. Check [effect.institute](https://www.effect.institute/?utm_source=cocoweb.fr) for learning it.
</ WebLinkSection>
